grid system

you have two mixins here for basic grid system the parent will has mixin (row) and the columns will have mixin (col).

.any_parent_class
   +row
   .any_child_class
      +col

the child here (col) will take the full width by default.

if you want to custome width (as any normal grid system), we make it easy for you the (+col) mixin has two variables. we assume you need (12) base grid system so we use it by default.

+col(6)

previous code will take 50% width

+col(1,2)

and this also will take 50% width

note that this effects will happen in all media typs (large screens to phone screen).

media queries

we have a defrrent way in using media querys we used large to small effect so what happens in large screen and normal code will still in small screen unliss you change it in that small screens.

you will find 4 main media break points we make it as mixins.

.any_class_name
   background: pink
   +col(1,2)
   +medium
      +col(5,12)
   +tablet
      +col(1,3)
   +phone
      +col

simply every media have its break point and you type your custome code under the mixin.

Note: what happens in tablet media will affect the tablet and phone media only. the effects goes in upside down way.

buttons

simply use the (button) mixin to make button effect with the default settings.

+button
+button(color, font color, font size, width, height)
+button /* default style */
/* custome style */
+button(#11CBD7, #FFF6A2, '30px', '2em', '40px')

Note: sizes can written in px, em, rem and %.
